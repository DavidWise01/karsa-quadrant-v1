# GT-MESH Useful AI Console v3.0

This turns the symbolic mesh into a practical AI orchestration console.

Open `index.html`.

Use it to generate structured packets for an AI session:

- Goal
- Constraints
- Current state
- Witness notes
- Drift risks
- Next action
- Mesh mode
- Containment/wave ratio
- Exportable JSON packet

The symbolic mesh becomes a working control surface for continuity and anti-drift.
